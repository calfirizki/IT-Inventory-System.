# IT-Inventory-System.
Sistem inventaris aset IT (Cisco) menggunakan Python dan MySQL

Deskripsi Proyek
Proyek ini adalah sistem manajemen inventaris aset IT yang dibangun untuk melacak dan mengelola infrastruktur teknologi, khususnya pada perangkat jaringan dan periferal. Sistem ini mengintegrasikan database MySQL dengan scripting Python untuk mempercepat proses input data, standarisasi kategori, hingga pelaporan profesional.

Proyek ini mendemonstrasikan kompetensi teknis saya dalam mengelola aset IT kritikal seperti Cisco Router dan Cisco Switch, yang sangat penting untuk pemeliharaan infrastruktur jaringan di lingkungan profesional.

Fitur Utama
Manajemen Database: Menggunakan MySQL untuk menyimpan informasi aset secara detail, termasuk nomor seri, status, dan lokasi (seperti Server Room Lt.1).

Standarisasi Data: Menggunakan logika SQL untuk memastikan kategorisasi aset yang tepat antara "Network" dan "Peripheral".

Ekspor Otomatis: Fungsi ekspor berbasis Python yang mengubah data mentah dari database menjadi file CSV/Excel yang terformat.

Laporan Visual: Dashboard Excel profesional yang menampilkan Pie Chart untuk memvisualisasikan distribusi kategori aset IT.

Output Siap Cetak: Tata letak laporan yang dioptimalkan dan diekspor ke format PDF untuk keperluan tinjauan manajemen.

Teknologi yang Digunakan
Bahasa Pemrograman: Python

Database: MySQL (Lingkungan XAMPP)

Pelaporan: Microsoft Excel & CSV

Tools: VS Code, Git/GitHub
